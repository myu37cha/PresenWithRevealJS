## 概要

このリポジトリはReveal.jsプレゼンテーションを作成・共有するためのテンプレートです。

## Reveal.jsとは

[Reveal.js](https://revealjs.com/)はHTML/CSS/JavaScriptで美しいプレゼンテーションを作成できるフレームワークです。

## 使い方

### スライドの追加

1. このリポジトリをフォークします
2. 新しいHTMLファイルを作成します
3. Reveal.jsを使用してスライドを作成します
4. プルリクエストを送信します

### スライドの閲覧

GitHub Pagesで公開されたURLにアクセスしてプレゼンテーションを閲覧できます。
